Switch statusline icon mode. Modes: "emoji" (default, works everywhere) or "nerd" (Nerd Font glyphs, requires terminal font set to a Nerd Font). Toggles if no argument given.

Run this command via Bash, replacing INSTALL_DIR with the directory containing statusline.sh (read from ~/.claude/settings.json statusLine.command path):

INSTALL_DIR/switch-icons.sh $ARGUMENTS

Report the output to the user.
